# 出题测试提示词

> 用法：对 Claude 说「出 {{题数}} 道 {{领域}} 题」，如「出 5 道敏捷题」。

## 提示词

请从 `04-practice/` 抽取或仿造 **{{题数}}** 道 **{{领域}}** 题考我。

要求：
- 优先用 `04-practice/{{领域对应目录}}` 真题。
- 一次只出一题，等我答完再出下一题并给解析。
- 记录对错，结束后汇总正确率。
- 错题自动记入 `05-mistakes/{{领域}}.md`，并更新 `00-dashboard`。

领域对应：
- 人员→people / 过程→process / 业务→business / 敏捷→agile / 综合→mock
