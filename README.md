## 一个月备考建议路径

1. **打基础（前 2 周）**：`02` 课件 + `01` 教材过一遍，同步看 `03/核心笔记` 和 `03/思维导图` 建立框架。
2. **分章刷题（第 3 周）**：`04/分章节练习` 按章节刷，错题回 `03` 查漏；敏捷弱的加刷 `04/敏捷专项`。
3. **模考冲刺（最后 1 周）**：`04/模拟与真题` 计时模考，吃透解析；`05/考前10天每日冲刺` 每天一套；考前看 `05/应试技巧` 的蒙题大法。
4. **官方题必刷**：`04/官方StudyHall` 是 PMI 官方题，风格最接近真实考试，优先做。

## 整理说明

原资料按"机构来源"分目录（希赛/骐迹/studyhall/小红书/2026含赠品），同一份资料散落多处。本次改为按内容类型重组，并分两轮去重：

**第一轮（md5 哈希严格比对，删 35 个字节完全一致的副本）**
- 23 个带 `(1)` 的下载重复副本（希赛课件全套等）
- 10 个跨目录同内容文件（三色笔记、知识点300条、180道真题、纯英文400道等）
- 2 个已被解压的冗余 `.rar`

**第二轮（同书不同扫描版，按页数核实为同一本书后各留 1 份干净版，删 4 个）**
- PMBOK第七版中文版：2 个扫描版 -> 留 1
- 敏捷实践指南中文版：3 个版本（含标记重点版）-> 留 1 干净版
- 过程组实践指南中文版：中文版 + 第八版划重点版 -> 留 1 干净版
- 英文版一律保留。

**第四轮（题目转 Markdown，可搜索/可标记错题）**
- `04-习题题库` 题目转成 md：题干+选项+答案+解析结构化，可 grep 搜索、可标错题。
- 自动解析多版式（题目/答案分文件配对、内联答案、xlsx 表格、`A：`/`A、`/`A` 多种选项符、中英文 `问题#N`/`Question #N` 题号）。
- 用 pdftotext 提取（比 OCR 更准，修复 studyhall 字体乱码和乐凯文字乱序），163 个 md，质量好的删原文件只留 md。
- 转不动的保留原件：docx 选项混排、纯英文特殊格式、答案图片版、纯题目文件（答案在姊妹文件）。
- 详见 `04-习题题库/转换记录.md`。

**其他**
- 同名但内容不同的版本已保留：`PMP重要考点10页纸.docx`、`PMP敏捷试题50道.docx` 各两版（已标注"另一版本"）。
- 视频文件原本未下载到，已按要求清空视频占位夹；保留的 19 份讲义课件是 PDF 不是视频。

**第三轮（压缩扫描图型 PDF，省 288M）**
- 大文件多为"整页扫描图片"型 PDF（每页 300~800KB），是体积主因。
- 用 ghostscript 将 37 个 >5MB 的 PDF 降到 100 DPI（屏幕阅读清晰、页数完整保留），759M -> 471M。
- 如某个 PDF 在你设备上看着偏糊，告诉我文件名，可单独按更高 DPI 重新压缩。
