# concepts · 长期知识资产（提炼版）

> 双层笔记的**资产层**：**Claude / 你提炼**的结构化知识点 md（不是教辅 PDF）。
> 与本目录的 PDF 教辅区别：PDF 是别人写的参考资料；这里的 .md 是学完后自己提炼的话，**可 grep 复习**。

## 何时写
- 学完一个模块后，Claude 把 `02-learning/<领域>/` 的节级笔记提炼成稳定知识点，写到这里。
- 命名 `<主题>.md`，如 `价值传递系统.md`、`12原则.md`、`8绩效域概览.md`、`Scrum角色工件事件.md`、`裁剪.md`。

## 与 02-learning 区别
- `02-learning/`：过程笔记，按节 / 学习顺序，记当时怎么学的、易错点。
- `concepts/`（这里）：资产，按主题，提炼后的稳定结论，复习时直接 grep。

## 复习时怎么用
- grep 一个关键词（如「裁剪」「干系人」），先看这里的 .md（你的话），再翻 PDF 教辅补细节。
